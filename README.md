## Structure

```text
config/
├── images/              # Brand assets
├── logo/                # SATUCHAIN logo
└── tokenlist/
    ├── satumainnet/     # Mainnet (Chain ID: 10111945)
    ├── satutestnet/     # Testnet (Chain ID: 17081945)

```
